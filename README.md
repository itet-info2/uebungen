### a

Bei x >= 0, wird "gerade" x mal aufgerufen.

Bei x >= 0, wird "verdopple" x mal aufgerufen.

Bei x >= 0 und x%2=1, wird "halbiere" (x-1)/2 mal aufgerufen.

Bei x >= 0 und x%2=0, wird "halbiere" x/2 mal aufgerufen.
